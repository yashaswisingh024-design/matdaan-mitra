# 🗳️ Matdaan Mitra

### Empowering Citizens Through Election Awareness

Matdaan Mitra is a modern, interactive web platform designed to simplify the Indian election process for citizens. The project aims to promote voter awareness, civic participation, and informed decision-making by presenting election-related information in a clear, engaging, and accessible format.

---

## 🌟 Overview

Voting is one of the most important rights and responsibilities in a democracy. However, many citizens, especially first-time voters, often find the election process confusing.

Matdaan Mitra bridges this gap by providing a user-friendly platform that explains how elections work in India, guides users through voter registration, and encourages informed participation through interactive features.

---

## 🚀 Features

### 📚 Election Process Guide

Learn the complete election journey in India, including:

* Voter Registration
* Electoral Roll Verification
* Polling Booth Information
* Electronic Voting Machines (EVMs)
* Vote Counting & Results
* Model Code of Conduct (MCC)

### ✅ Voter Registration Checker

A simulated voter verification tool that helps users understand how registration checks work.

### 🧠 Interactive Election Quiz

Test your knowledge of:

* Voting rights
* Election procedures
* ECI guidelines
* NOTA and EVMs
* Democratic participation

### 📊 Candidate Comparison

Compare sample candidates based on:

* Education
* Experience
* Party Affiliation
* Key Promises

### ❓ Frequently Asked Questions

Quick answers to common election-related queries.

### 📱 Fully Responsive Design

Optimized for:

* Desktop
* Tablet
* Mobile Devices
