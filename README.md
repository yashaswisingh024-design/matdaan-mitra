# Matdaan Mitra (VoteSmart) 🗳️🇮🇳

**Matdaan Mitra** is a clean, responsive, and beginner-friendly web application designed to educate citizens about the election process in India. By breaking down complex democratic procedures into easy-to-understand steps, the app aims to empower voters and increase civic participation.

---

## 📖 Description

Elections in the world's largest democracy can sometimes feel overwhelming for first-time voters. Matdaan Mitra serves as a digital guide, taking users step-by-step through the entire electoral journey—from voter registration to casting a vote on an Electronic Voting Machine (EVM). It acts as a comprehensive, visually appealing educational platform for Indian citizens.

---

## ✨ Features

- **Educational Walkthrough:** A step-by-step guide explaining Voter Registration, the Electoral Roll, Polling Booths, EVM Voting, and the Model Code of Conduct.
- **Candidate Comparison:** A responsive, structured table allowing voters to objectively compare candidates based on Party Affiliation, Education, Key Promises, and Experience.
- **Interactive Election Quiz:** A dynamic multiple-choice quiz to test users' knowledge of the Indian electoral system, featuring real-time validation, score calculation, and answer highlighting.
- **Voter Registration Simulator:** An interactive form that allows users to simulate checking their EPIC (Voter ID) status.
- **Google Maps Integration:** A quick-access button that utilizes Google Maps to help users locate polling booths near their current location.
- **Google Forms-style Feedback:** A built-in, anonymous feedback form with a custom interactive star-rating system.
- **Modern UI/UX:** Built with premium design aesthetics, including micro-animations, glass-morphism effects, scroll-reveal transitions, and full mobile responsiveness.
- **Accessible Design:** Developed with Web Content Accessibility Guidelines (WCAG) in mind, featuring clear contrast, ARIA labels, semantic HTML, and proper keyboard focus states.

---

## 🛠️ Tech Stack

This project is built using pure vanilla web technologies to ensure maximum performance, modularity, and zero dependencies:

- **HTML5:** For semantic structuring and accessible content delivery.
- **CSS3:** For modern styling, CSS variables (design tokens), flexbox/grid layouts, and responsive media queries.
- **JavaScript (ES6):** For handling DOM manipulation, form validation, quiz logic, and interactive components.

---

## 🚀 How It Works

1. **Clone or Download the Repository:** Save the project files to your local machine.
2. **Open the App:** Simply double-click on `index.html` to open the app directly in your default web browser. No server setup, Node.js, or compilation is required.
3. **Navigate:** Use the sticky navigation bar to explore different sections:
   - Read the **Steps** to understand how voting works.
   - Use the **Compare** tab to evaluate sample candidates.
   - Test your knowledge with the **Quiz**.
   - Find your polling booth using the **Check Registration** tool.

---

## ⚠️ Assumptions

- **Educational Purpose:** The voter ID check is a frontend simulation meant for educational purposes. It does not connect to the actual Election Commission of India (ECI) backend database. Users are directed to the official ECI portal for real verification.
- **Sample Data:** The Candidate Comparison table currently uses hardcoded placeholder data.
- **Maps Link:** The Google Maps "Nearest Polling Booth" button uses a general search query (`polling booth near me`) rather than relying on a complex API implementation.

## 🔮 Future Improvements

- **API Integration:** Connect the app to civic APIs to pull real-time data on upcoming elections, candidate profiles, and official electoral rolls.
